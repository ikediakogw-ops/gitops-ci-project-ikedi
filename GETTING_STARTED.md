# Getting Started

## Setup

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

## Run Tests

pytest test_app.py -v

## Branch Workflow

- Create branch: feature/TRELLO-###
- Make changes
- Commit with Trello ID
- Push and create PR

## Create PR

- Go to GitHub
- Open Pull Request
- Wait for CI to pass
- Merge

## Troubleshooting

- If tests fail → fix code
- If lint fails → fix formatting
